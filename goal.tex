\section{目標}

% 野村の担当

今回我々が目指したのは，以下の機能をもつAMラジオである。
\begin{enumerate}
  \item ゲルマニウムダイオードを用いた検波を行うことができる。(ゲルマラジオ)
  \item アンプを用いてスピーカーを駆動することができる。
  \item バリコンを用いて選局を行うことができる。
  \item 可変抵抗を用いて音量調整を行うことができる。
  \item 持ち運びできるサイズである。
  \item 部屋のどこでも受信できる。
\end{enumerate}
